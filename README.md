# FGAesthetic NFC Loyalty System
**Stack:** React · TypeScript · Vite · Tailwind · Supabase


  - **Project root:** `/`
  - **Frontend:** `src/` (Vite + React)
  - **Supabase client:** `src/supabaseclient.ts`


    install dep; run dev server(vite):

    ```cmd
    npm install
    npm run dev
    ```



    `http://localhost:5173` local server

    **prod build**

    ```cmd
    npm run build
    npm run preview
    ```
